# push_swap - Dealing with Sorting Complexity

## Description

My implementation of one of the projects of the 42 school, which, given two stacks, a random list of integers and a limited set of instructions, creates a program that must sort the integers with the smallest one at the top of the first stack, and with the lowest number of executed instructions. A lesson in sorting complexity.
